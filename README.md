# 🔐 CryptoShit

Многофункциональный криптографический инструмент с графическим интерфейсом на Python.

Проект предназначен для изучения современных криптографических алгоритмов, генерации ключей, вычисления хэш-функций и работы с различными источниками энтропии.

> ⚠️ **Проект создан исключительно в образовательных и исследовательских целях.**

---

# Возможности

## Симметричное шифрование

* AES (Fernet)
* ГОСТ 34.12-2015 «Кузнечик»

## Асимметричное шифрование

* RSA (2048+ бит)

## Потоковое шифрование

* ChaCha20-Poly1305

## Хэш-функции

* SHA-256
* SHA-512
* SHA3-256
* BLAKE2b
* ГОСТ Р 34.11-2012 «Стрибог-256»
* ГОСТ Р 34.11-2012 «Стрибог-512»

## Генерация энтропии

Поддерживаются различные источники случайности:

* движение мыши;
* шум микрофона;
* шум матрицы веб-камеры;
* системная энтропия (таймеры, загрузка процессора, память и др.).

## Дополнительные возможности

* генерация и сохранение ключей;
* загрузка ранее сохранённых ключей;
* установка срока действия зашифрованных данных;
* просмотр hex-дампа собранной энтропии;
* сохранение сырой энтропии в файл;
* удобный GUI на Tkinter.

